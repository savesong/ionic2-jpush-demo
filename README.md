# Ionic2 jPush Demo
基于最新版本ionic2，集成极光推送，做了两个常用示例：

1. 获取Registration ID。
2. App接收到推送通知后跳转到特定页面。

## 注意事项
由于jPush是集成到ionic-native，需要用此项目的node_modules/ionic-native/dist替换你的ionic2项目对应目录的内容。
